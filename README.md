🔴 My_Pc_Pokémon
Projeto SENAI | Sistema CRUD inspirado na interface de gerenciamento de Pokémon dos jogos clássicos.

Este projeto recria a experiência do icônico "PC do Bill" (ou PC de Celadon), introduzido em Pokémon FireRed/LeafGreen. O sistema permite o gerenciamento completo (Create, Read, Update, Delete) de uma coleção de Pokémon, unindo a nostalgia dos anos 2000 com práticas modernas de desenvolvimento de software.

🚀 Funcionalidades
Capturar Pokémon (Create): Adicione novos monstrinhos ao seu sistema informando nome, tipo, level e status.

Visualizar Box (Read): Liste todos os Pokémon armazenados, com filtros por tipo ou raridade.

Treinar/Editar (Update): Atualize as informações de um Pokémon já registrado.

Libertar (Delete): Remova um Pokémon do sistema (a famosa despedida para a natureza).

🛠️ Tecnologias Utilizadas

Linguagem: [JavaScript]

Interface: [HTML & CSS]

Banco de Dados: [ MySQL / JSON]

Arquitetura: Padrão CRUD (Create, Read, Update, Delete).
